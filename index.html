PrestaShop + Koding.com

Installez, hébergez et jouez avec la version PrestaShop v1.6-alpha

Temps nécessaire : 7 min • Niveau : Facile

  • Inscrivez-vous sur Koding

    Koding fournit gratuitement un service puissant pour développer sur un serveur virtuel à partir de votre navigateur.

    OK, c'est parti !

  • Quel est votre nom d'utilisateur Koding ?

  • Accédez à l’onglet terminal de Koding

    L'onglet terminal va vous permettre d'intéragir avec votre machine virtuelle en saisissant des lignes de commandes.

    Accédez au Terminal

  • Clonez et installez PrestaShop v1.6 sur votre nouvelle machine virtuelle

    Il vous suffit de copier/coller et executer chacune de ces lignes dans le terminal.
    Note : Sur Windows, le raccourci clavier CTRL+V ne fonctionne pas à partir du terminal : utilisez le menu du clic droit.

    • git clone https://github.com/PrestaShop/PrestaShop.git -b 1.6 Web/PS16 Copier
    • cd Web/PS16 Copier
    • chmod -R 777 cache/ config/ download/ img/ log/ mails/ modules/ themes/*/lang themes/*/cache themes/*/pdf/lang/ translations/ upload/ -f Copier
    • php install-dev/index_cli.php --domain={{username}}.kd.io --db_create=1 Copier

  • Félicitations !
