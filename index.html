PrestaShop + Koding.com

Installez, hébergez et jouez avec la version alpha PrestaShop v1.6

  • Inscrivez-vous sur Koding.com

    Le plus facile est d'utiliser votre compte Github.

    Koding fournit gratuitement un service puissant pour développer sur un serveur virtuel à partir de votre navigateur.

  • Quel est votre nom d'utilisateur Koding ?

    Verifier

  • Accédez à l’onglet terminal de Koding

    L'onglet terminal va vous permettre d'intéragir avec votre machine virtuelle en saisissant des lignes de commandes.

    Accédez au terminal

  • Clonez et installez PrestaShop v1.6 sur votre nouvelle machine virtuelle

    • git clone https://github.com/PrestaShop/PrestaShop.git -b 1.6 Web/PS16
    • cd Web/PS16
    • chmod -R 777 cache/ config/ download/ img/ log/ mails/ modules/ themes/*/lang themes/*/cache themes/*/pdf/lang/ translations/ upload/ -f
    • php install-dev/index_cli.php --domain={{username}}.kd.io
