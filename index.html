Step by Step Guide

  • Identifiez-vous sur Koding.com via votre compte Github
  • Ouvrez l’onglet du terminal git clone https://github.com/PrestaShop/PrestaShop.git Web/PS16 cd Web/PS16/PrestaShop chmod -R 777 cache/ config/ log/ img/ mails/ modules/ themes/*/lang themes/*/cache translations/ upload/ download/ themes/*/pdf/lang/ -f

Ensuite vous pouvez directement accéder à l’url d’installation de votre boutique : http://[USERNAME].kd.io/PS16/install-dev/ et suivez le processus d’installation de votre nouvelle boutique. (PrestaShop est capable de créer sa table de données tout seul)
